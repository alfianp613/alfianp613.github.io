Cara Membuat Github Page

28 Oktober 2021

Halo semuanya! Pada blog pertamaku, aku akan membahas mengenai cara membuat Github Pages menggunakan sistem Git.

Mungkin kamu memiliki file web statis yang memuat portfolio kalian atau blog pribadi kamu dan kamu ingin meng-hosting web kamu secara gratis agar dapat diakses oleh orang di seluruh Dunia secara gratis, maka Github Pages adalah solusinya.

Github Pages merupakan layanan hosting website gratis dari Github. Seluruh file website akan di-hosting melalui GitHub, artinya apabila kamu ingin melakukan perubahan kamu bisa melakukannya melalui github atau langsung dari desktop melalui perintah-perintah git.
